## Role Overview

Project Assistants handle client intake, admissions processing, student attendance, and activity coordination. Your role serves as the primary point of contact for new clients and manages the initial stages of the student journey from registration through admissions.

## What You Can Access

### **Register & Inquiry** - View & Add/Edit Client Information
- Register new clients in the system
- Record client inquiries and follow-up
- Edit existing client information
- Search and manage client database

### **Admissions** - View & Add/Edit Applicant Processing
- Add new applicants from registered clients
- Input interview scores and assessments
- Process admission applications
- Manage applicant status and progression

### **Student Records** - View Only
- View student information and enrollment details
- Access student academic progress
- Review student status and course assignments
- Cannot edit or modify student records

### **Student Attendance** - View, Add Sessions, Edit Attendance
- View attendance records for all classes
- Create new attendance sessions (maximum 4 per week)
- Mark student attendance as Yes/No/Excuse
- Edit existing attendance entries

### **Volunteer Time Record** - View & Add Records
- View volunteer time records
- Add new volunteer time entries
- Submit volunteer hours
- Track volunteer activity participation

### **Activities** - View & Add/Edit Events and Participants
- Create and manage activities and events
- Add participants to activities
- Coordinate activity schedules
- Track event attendance and participation

## Your Common Workflows

### Client Registration and Inquiry Process

1. Navigate to **Register & Inquiry** → **Register**
2. Search for existing clients to avoid duplicates
3. Click "Add New Client" if not found
4. Fill in required client information:
   - Personal details
   - Contact information
   - Background information
5. Save the client record (generates URPID)
6. Record initial inquiry in **Inquiry** section
7. Schedule follow-up as needed

### Processing New Admissions

1. Go to **Admissions** → **Add Applicant**
2. Select from registered clients
3. Choose appropriate course and level
4. Schedule interview appointment
5. After interview, input scores in **Input Interview Score**
6. Determine pass/fail status
7. Successful applicants automatically progress to Student Records

### Managing Student Attendance Sessions

1. Navigate to **Students** → **Student Attendance**
2. Select "Create Date Session" to start new session
3. Choose class, date, and time
4. Set session parameters (max 4 per week per class)
5. Mark attendance for each student:
   - **Yes** - Present
   - **No** - Absent
   - **Excuse** - Excused absence
6. Save attendance session
7. Edit previous sessions if corrections needed

### Coordinating Activities and Events

1. Access **Activities** menu
2. Click "Add Activity" to create new event
3. Fill in activity details:
   - Activity name and description
   - Date, time, and location
   - Activity type and category
4. Add participants:
   - Select from student database
   - Add external participants if needed
5. Assign facilitators (staff or volunteers)
6. Track attendance and participation
7. Update activity status as needed

### Recording Volunteer Time

1. Access **Volunteers** → **Volunteer Time Record**
2. Click "Add Record" or find existing entry
3. Complete required fields:
   - Stipend base
   - Time records (hours worked)
   - Meetings attended
   - Hourly rate (if applicable)
4. Submit time record for processing

## Limitations & Notes

:::caution[What Project Assistants Cannot Do]
- **Cannot edit student records** - Contact EO, PO, or PD
- **Cannot manage GEL scores** - Contact Teachers or EO
- **Cannot manage scholarships** - Contact PO or PD
- **Cannot manage volunteers** - Contact EO, PO, or PD
- **Cannot access finance functions** - Contact AA or PD
- **Cannot delete records** - Contact PD
:::

### Need Help With Other Tasks?

- **Student record modifications** → Contact Education Officer (EO)
- **GEL test score entry** → Contact Teachers or Education Officer (EO)
- **Scholarship management** → Contact Project Officer (PO)
- **Volunteer management** → Contact Education Officer (EO)
- **Finance or payments** → Contact Admin & Finance Assistant (AA)
- **System administration** → Contact Project Director (PD)
